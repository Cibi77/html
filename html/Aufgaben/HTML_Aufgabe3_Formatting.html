Fett gedruckter Text mit <bold>

In diesem Text hat es fett gedruckten Text der unschwer zu erkennen ist.

Wichtige Informationen mit <strong>

Wichtiger Text oder wichtige Informationen können mit semantisch speziell ausgezeichnet werden. Auch wenn visuell kein Unterschied zu fett gedrucktem Text zu erkennen ist, hat er doch eine andere semantische Bedeutung.

Eine ganze Reiher weiterer semantisch bedeutender Tags

Text kann schräg (italic) ausgezeichnet werden.

Text kann betont (emphatized) ausgezeichnet werden.

Text kann markiert ausgezeichnet werden.
Text kann kleiner (small) ausgezeichent werden.
Text kann durchgestrichen (deleted) ausgezeichent werden.
Text kann eingefügt (inserted) ausgezeichent werden.
Text kann tiefgestellt (subscript) ausgezeichent werden.
Text kann hochgestellt (superscript) ausgezeichnet werden.
Code kann mit int = c1 dargestellt werden.
Text kann formatiert ausgezeichnet werden: 

        ....... 
           Leerzeichen werden übernommen.
     Tabulatoren werden übernommen.
